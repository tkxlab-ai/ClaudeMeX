# Releases

ClaudeMeX is distributed as a self-contained `.skill` bundle.
Download the latest from the [Releases page](../../releases).

Each release ships:
- `claudemex-vX.Y.Z.skill` — the installable skill bundle
- `checksums.txt` — SHA-256 of all assets
- `sbom.cyclonedx.json` — CycloneDX software bill of materials

Source tree is maintained in a private mirror; public distribution is
release-artifact only.
